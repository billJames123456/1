#### 使用技术及框架
服务端：

- Springboot2.7
- mysql 8.0
- mybatis 3.5.2
- redis:3.2(windows版) ,服务器liunx采用redis7.0
- 阿里云图像识别
- jjwt(请求token)
- thumbnailator(图片压缩工具)

前端：

- vue 2.6.7  elementui2.15.8 ,少量ant design 组件
- echarts.js
- axios
